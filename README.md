# MediaWiki Enhancer: A Better MediaWiki Experience

MediaWiki Enhancer improves how you interact with MediaWiki powered sites by adding useful capabilities that make everyday tasks smoother.

### Features

- **Smart Image Paste:** Paste an image into supported areas and the extension handles the upload process automatically.
- **More features coming soon.**

Want to see the source code or contribute? Check out the [GitHub repository](https://github.com/SahilAggarwal2004/mediawiki-enhancer).
